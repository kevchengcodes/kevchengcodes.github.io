ABOUT ME


Welcome to my page. I'm Kevin, and I am a software engineer at Swoop Search with a heavy focus on ML/AI. I'm currently working remotely in Seattle and having a great time building deep learning and PNT solutions into our edge computing platforms.

I joined Swoop Search in November 2020 after almost four years at Boeing. During my time there, I created predictive maintenance algorithms in Python/SQL, real-time simulations in MATLAB/Simulink, and flight guidance optimization software in C/C++.

This website!

I built this page from scratch (with a lot of Googling, of course) using HTML, CSS, and jQuery. You can check out the source code here


November 2020

 

I've always been interested in UI/UX and front-end design, so I knew I'd have a lot of fun building a webpage to showcase myself. This page started out as a basic collection of simple divs, paragraphs, and images, but as I got more into it, I built some functions that I personally think are pretty neat, such as:

The ability to collapse or expand sections (like this one) by assigning it an ID and writing a jQuery script for a button to control the display:

$(document).ready(function(){
   $("#expand-web").click(function(){
      $("#description-web").toggle();
      $('html, body').animate({
         scrollTop: $("#expand-web").offset().top
      }, 200);
   });
   $("#description-close-web").click(function(){
      $("#description-web").hide();
   });
};

Or changing the transparency of a background image without affecting the characteristics of the text in front of it, by assigning the opacity of the image in a pseudo-element:

.Blog-post::before {
   content:"";
   background-image:url('windmill.jpg');
   background-size: cover;
   position: absolute;
   top: 0px;
   right: 0px;
   bottom: 0px;
   left: 0px;
   opacity: 0.25;
}

Or creating a floating navigation bar by using jQuery to add a class imposing a top-level z-index and positioning at the top of the screen if the user scrolls down:

var nav=$('.nav-container');
   $(window).scroll(function(){
      if ($(this).scrollTop()>300){
         nav.addClass("f-nav");
      }
   else {nav.removeClass("f-nav");}
});

The best part of creating your own webpage is how creative you can be. You can dream up any features that you might want to incorporate, and chances are, you can code it.

WeMinari: Educational Web App

My first class in OMSCS at Goergia Tech was Educational Technology, where they challenged us to create a new tech to aid with teaching.


May 2021

 

After researching basic pedagogies, I decided to prototype a web app that doubles as an adaptive learning management system and a social media platform. This web application connects students and instructors to active learning opportunities outside of the classroom. Additional features include an individualized news feed, multiple discussion boards, and classroom management capabilities for the instructor.

The individualized news feed uses web scraping and natural language processing to present the user with personalized news articles that may interest them, based on their history. The web app backend is written in Python and is built on a Django framework. Data is stored in PostgreSQL databases and was hosted on Heroku back when there was a free tier.
